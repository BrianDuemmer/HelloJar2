How to run MicroLog in Ubuntu:
---------------------------------

1. Download libbluetooth-dev from the package manager.
2. Download bluecove.jar: http://snapshot.bluecove.org/distribution/download/2.1.1-SNAPSHOT/2.1.1-SNAPSHOT.60/bluecove-2.1.1-SNAPSHOT.jar
2. Download bluecove-gpl.jar: http://snapshot.bluecove.org/distribution/download/2.1.1-SNAPSHOT/2.1.1-SNAPSHOT.48/bluecove-gpl-2.1.1-SNAPSHOT.jar
3. Save microlog-server-bluetooth-[version].jar, bluecove.jar and bluecove-gpl.jar together with the start-microlog-server.sh in the same folder.
4. Run the start-microlog-server.sh shell script.

(The shell script will handle other versions of Bluecove, we recommend using the latest Bluecove version and no older than the 2.1.1 snapshot versions).
